Απορίες που αφορούν την 6η άσκηση
Ε: Ποιές μεθόδους θέλετε να υλοποιήσουμε(είναι αρκετές η put(Object key, Object value), get(Object key), containsKey(Object key), containsValue(Object value)=>functions που χρησιμοποίηθηκαν στο τελευταίο εργαστήριο;
Α: Ουσιαστικά απαραιτητη για την ασκηση είναι μονο μια put(int value) που τοποθετει στον πινακα κατακερματισμου ένα στοιχειο που εχει συγκεκριμενη τιμη έναν τυχαιο ακεραιο. Εσωτερικα, η κλαση που υλοποιει τον κατακερματισμο θα πρεπει να υπολογιζει σε ποια θεση του πινακα πρεπει να τοποθετηθει αυτό το στοιχειο (υπολογιζοντας τη συναρτηση κατακερματισμου για το value) και αν αυτή είναι κατειλημμενη, να ψαχνει τις επομενες. Όταν καταφερει να βρει καποια κενη, θα το εισαγει επιστρεφοντας το πληθος των συγκρουσεων. Όταν εκτελεστουν λ.Ν put θα υπολογιζετε το μεσο ορο των συγκρουσεων.

Ε: Μπορούμε να χρησιμοποιήσουμε οποιαδήποτε hash συνάρτηση h(k); Πρέπει να είναι η βέλτιστη;
Α: Αρχικα κάθε συναρτηση που επιστρεφει τιμες 0..Ν-1 είναι αποδεκτη. Θα εκτιμηθει αν διαλεξετε μια 'καλη' συναρτηση κατακερματισμου.

Ε: Θα κάνουμε βελτιστοποίση του ολικού προγράμματος;
Α: Πρωτη προτεραιοτητα είναι η αναγνωσιμοτητα και η αρθρωσιμοτητα.
